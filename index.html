Using GitHub pages to host static sites

GitHub can be used both for storing my profiles remotely and host my projects. This happens freely for public repositories. If your Repo is private, you will need to upgrade in order to host the project.

Steps

  1. Initialize git init in your project files directory
  2. Add all the project files
  3. Commit them using suitable messages
  4. Create a new repo in your GitHub account
  5. Push the existing repo from yor computer to the newly created GitHub Repo
  6. Go to GitHub pages settings and select branch and root directory then save
  7. You will be provided with a live link for your hosted project
  8. Save
